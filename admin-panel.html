What This Panel Controls

This admin panel now focuses on the actions that are fully connected: user roles, subscriptions, workspace cleanup, report handling, and announcements.

Users Search accounts, upgrade subscriptions, and change roles with hierarchy checks.
Workspaces Review created workspaces and remove invalid or abusive ones.
Reports Process moderation reports without leaving the page.
Announcements Publish site-wide updates for the rest of the platform.
